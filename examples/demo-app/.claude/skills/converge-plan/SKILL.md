# converge-plan

Generated by ConvergeKit from .converge/attractor.yml.

Run:

```bash
converge plan "<task title>" --type <bugfix|feature|refactor>
```

Then fill in Goal, Non-goals (protected paths), and Exit Criteria in the generated plan file before implementing.
